# ciencia-de-datos-unab
Proyecto de ejemplo para materiales y ejercicios de introducción a Ciencia de Datos.

Descripción

Repositorio que contiene notebooks, datos y scripts usados en prácticas y proyectos.
Estructura

README.md : este archivo.
notebooks/ : cuadernos Jupyter con análisis y ejercicios.
data/ : datos de ejemplo (si aplica).
scripts/ : scripts y utilidades en Python.
Requisitos

Python 3.9 o superior
Paquetes típicos: pandas, numpy, scikit-learn, matplotlib, seaborn, jupyter.
Instalación rápida

Crear y activar un entorno virtual:
python -m venv venv
source venv/bin/activate
Instalar dependencias (si existe requirements.txt):
pip install -r requirements.txt
Uso

Abrir los notebooks con JupyterLab o Jupyter Notebook:
jupyter lab
